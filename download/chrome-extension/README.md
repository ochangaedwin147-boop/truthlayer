# TruthLayer Chrome Extension

AI-powered misinformation detection right in your browser.

## Installation

### From Source (Developer Mode)

1. Download or clone this `chrome-extension` folder
2. Open Chrome and navigate to `chrome://extensions/`
3. Enable "Developer mode" in the top right corner
4. Click "Load unpacked" and select the `chrome-extension` folder
5. The TruthLayer extension icon will appear in your toolbar

## Configuration

1. Click the TruthLayer icon in your browser toolbar
2. Click the ⚙️ settings icon
3. Enter your API URL (e.g., `https://your-truthlayer-instance.com`)
4. Enter your API key from your TruthLayer dashboard
5. Click "Save"

## Usage

### Text Mode
1. Click the extension icon
2. Paste any text content into the text area
3. Click "Verify Content"
4. View the trust score and analysis

### Selection Mode
1. Select text on any webpage
2. Click the extension icon
3. Switch to "Selection" tab
4. Click "Get Selection"
5. Click "Verify Content"

## Features

- **Trust Score**: 0-100 score indicating content reliability
- **Warning Flags**: Automatic detection of potential issues
- **Detailed Analysis**: AI-powered explanation of the score
- **Content Hash**: Unique identifier for verified content
